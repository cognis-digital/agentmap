# Security Policy

## Reporting a vulnerability

Please report security issues privately to **security@cognis.digital**.
Do not open a public GitHub issue for vulnerabilities.

We aim to acknowledge reports within 3 business days and to provide a
remediation timeline after triage.

## Scope

`agentmap` is dual-use security software. It reads local configuration and
emits a graph + findings; it makes no network calls and executes none of the
servers it maps. Use it only against systems, data, and identities you own or
are explicitly authorized in writing to test, in compliance with applicable law.
